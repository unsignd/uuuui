# useTheme()

`useTheme()` is a hook used to access the getter and setter of library's theme. You can load the current theme and set the theme with the `theme` and `setTheme()`.

```tsx
import { useTheme } from '@unsignd/uuuui';

function Component() {
  const { theme, setTheme } = useTheme();
  //...
```

## Usage

<br />

#### `theme`: [ThemeType](/docs/type-themetype--docs)

The `theme` variable holds the current theme.

```tsx
console.log(theme);
```

<hr />

#### `setTheme()`: (theme: [ThemeType](/docs/type-themetype--docs))) => void

The `setTheme()` function allows you to change the theme. Check the [Theme switching example](/story/hook-usetheme--theme-switch).

```tsx
setTheme('light');
setTheme('dark);

// Switch theme.
setTheme(theme === 'light' ? 'dark' : 'light');
```

## Initial Value

If the component is wrapped with `<Provider>`, it automatically sets the `theme`'s initial value below unless you provide a `theme` property.

```json
"light"
```

**uuuui** is an open-source React Typescript component library.<br/>
Anyone can use **uuuui** by installing it using `npm` or `yarn`.

<br />

## Install

To install `uuuui`, run one of the following commands on your terminal.

```bash
# NPM
npm i @unsignd/uuuui

# YARN
yarn add @unsignd/uuuui
```

## Setup

To make `uuuui` work properly, a `<Provider>` component should wrap other `uuuui` components.
